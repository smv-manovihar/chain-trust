# Route: /manufacturer (Manufacturer Dashboard)

The Manufacturer Dashboard is the command center for production tracking and security alerts.

## Layout Overview
- **KPI Cards**:
  - **Total Products Enrolled**
  - **Active Batches**
  - **Global Scans (24h)**
  - **Unread Security Alerts**
- **Quick Actions**:
  - **Add Product SKU**: Redirects to `/manufacturer/products/new`.
  - **Enroll New Batch**: Redirects to `/manufacturer/batches/new`.
  - **Security Analytics**: Redirects to `/manufacturer/analytics`.
  - **Manage Team**: Redirects to `/manufacturer/team`.
- **Top 5 Alerts**: Real-time ticker of the latest security or verification events.
- **Production Analytics**: A chart/visual representation of scan trends (Authentic vs. Suspicious).

## AI Context
This page provides the most balanced overview for a manufacturer. AI should use this to summarize general production health and alert status.
